나의 미래 이야기 앱 (3·4차시 통합)

실행 방법
1. index.html을 더블 클릭합니다.
2. 첫 화면에서 활동을 고릅니다.
   - 3차시: 미래의 나를 만들어요
   - 4차시: 나를 소개해요
   - 연습 활동: 복사 붙여넣기 연습

3차시 흐름
그림 선택 → AI 문장 만들기 → 복사 → 생성형 AI에 붙여넣기

4차시 흐름
별명·학년·넓은 지역·좋아하는 것 입력 → 제미나이 질문 만들기 → 복사 → 제미나이에 붙여넣기 → 생성된 자기소개를 구글 홈페이지에 붙여넣기

복사 붙여넣기 연습 흐름
짧은 글자 드래그 → 긴 글자 드래그 → 드래그 후 Ctrl+C → Ctrl+C 후 Ctrl+V → 확인하기

개인정보 안전
- 실명 대신 별명을 권장합니다.
- 지역은 시·도 정도만 입력합니다.
- 학교명, 상세주소, 생년월일, 전화번호는 입력하지 않습니다.
- 자세한 지역 정보는 제미나이 질문에 포함되지 않습니다.

교사용 메모
- 자기소개 질문은 이름 또는 별명만 필수이며 나머지는 비워도 됩니다.
- 제미나이 사용이 어려운 학생은 결과 화면의 ‘바로 쓰는 자기소개’를 이용합니다.
- 좋아하는 것을 글로 쓰기 어려우면 그림 버튼 하나만 선택해도 됩니다.
- 1차시 결과 JSON을 불러오면 미래 이미지의 캐릭터와 직업활동이 먼저 선택됩니다.
- 각 결과는 별도의 복사 버튼과 텍스트 파일로 저장할 수 있습니다.
- 복사 붙여넣기 연습은 10단계로, 1-4단계는 드래그, 5-7단계는 드래그 후 Ctrl+C, 8-10단계는 Ctrl+V 붙여넣기까지 연습합니다.
